# ✨ Features

This document provides an overview of all major functionalities implemented in the DVLD (Driving & Vehicle License Department) Management System.

---

# Authentication

- Secure user login
- User authentication
- User authorization based on permissions

---

# People Management

Manage all people registered in the system.

### Features

- Add new person
- Update person information
- Delete person
- Search people
- Filter records
- View person details

---

# User Management

Manage system users.

### Features

- Add new user
- Update user
- Delete user
- Activate / Deactivate user
- Search users

---

# Driver Management

Manage registered drivers.

### Features

- View drivers
- Search drivers
- Driver history
- License history

---

# Local Driving License Applications

Create and manage local driving license applications.

### Features

- Create application
- Select license class
- Track application status
- View application information

---

# Driving Tests

Applicants must successfully complete all required tests.

### Vision Test

- Schedule vision test
- Take vision test
- Record result

### Written Test

- Schedule written test
- Take written test
- Record result

### Street Test

- Schedule street test
- Take street test
- Record result

---

# Driving License Services

The application supports several license services.

### First-Time License

Issue a driving license after all required tests are passed.

### License Renewal

Renew an expired driving license.

### Replacement Services

- Replace lost license
- Replace damaged license

---

# International Driving License

Issue international driving licenses for eligible drivers.

### Features

- Create application
- Issue international license
- Search international licenses
- View license information

---

# License Detention

Manage detained licenses.

### Features

- Detain license
- Release detained license
- View detained licenses

---

# Application Management

The system maintains complete application records.

### Features

- View application history
- Search applications
- Filter applications
- Track application status

---

# Search & Filtering

Most management screens provide:

- Search by ID
- Search by Name
- Search by National Number
- Custom filters

---

# Data Validation

The application validates data before saving.

Examples include:

- Required fields
- Duplicate national number
- Duplicate username
- Application eligibility
- License eligibility
- Test prerequisites

---

# Summary

The DVLD system manages the complete lifecycle of a driving license, from applicant registration to license issuance and subsequent services such as renewal, replacement, detention, release, and international licensing.
